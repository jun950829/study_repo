# # 컴퓨터 구조

### 프로그램을 이루는 두 정보 - 명령어, 데이터
모두 0과 1로 이루어진 정보

### 컴퓨터의 네 가지 핵심 부품

#### 1. CPU
- 명령어 해석, 실행 담당하는 장치

  - 레지스터 : CPU 내부의 작은 임시 저장 장치
  - ALU : 계산기 회로
  - 제어 장치 : 명령어를 해석, 제어 신호 내보내기

#### 2. 메인 메모리 & 캐시 메모리
- 실행 중인 프로그램 저장하는 장치 ( 전원이 꺼지면 삭제 )
  - 메인 메모리에서는 주소라는 개념이 이용됌

#### 3. 보조 기억 장치
- 보관할 프로그램 저장하는 장치 ( 정원이 꺼져도 저장 유지 )

#### 4. 입출력 장치
- 컴퓨터 내외부와 정보를 주고 받는 장치

#### 5. 메인보드
- 핵심 부품을 연결할 기판
- ( 시스템 )버스: 부품간 정보를 주고 받을 통로
